The files in the directory:
	- CrN.cif - original structure of CrN from https://materialsproject.org/materials/mp-1000440/
	- parse_utils.py - bunch of utils functions for parsing output directories
	- utils.py - bunch of file r/w utilities

** For specific use please refer to docstrings in each file. **

REQUIREMENTS:
	- pymatgen: https://pymatgen.org/
	- argparse: https://docs.python.org/3/library/argparse.html
